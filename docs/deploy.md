## Deployment & Production Plan

### 1. Docker Image
With the Dockerfile makes the app easily deployable in any containerazied environment (k8s, ECS, etc). 

#### 1. Build the container: 
```sh
$ docker build docker build -t tiqets-assignment .
```

#### 2. Run the container:
```sh
$ docker run --rm tiqets-assignment
```
This command will run the script with default arguments.

If you want to see the output CSV from inside the container you can use the following command:
```sh
docker run --rm --entrypoint sh tiqets-assignment -c "python src/main.py --output ./data/output.csv && cat /app/data/output.csv"
```
This command will print on terminal the contents of `output.csv` generated by the script.

#### 2. Monitoring
The code will exit with code `1` if any error occurs. That way we can monitor the container exit code and generate alerts on top of it to be alerted 
when something goes wrong.
